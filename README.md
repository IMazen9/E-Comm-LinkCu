🛍️ E-Comm-LinkCu

A modern front-end e-commerce website that simulates an online fashion store specializing in semi-formal clothing. The project is built entirely with HTML5 and CSS3, making it lightweight, responsive, and easy to deploy as a static website.

📖 Overview

E-Comm-LinkCu is designed to provide a clean and engaging shopping experience for a Cairo-based fashion retailer. It demonstrates responsive web design principles, consistent branding, and intuitive navigation without relying on JavaScript frameworks or backend technologies.

✨ Features
🏠 Responsive Home page with featured products and promotional banners
🛒 Shop hub with separate Men's and Women's collections
👔 Product catalog pages with pricing displayed in Egyptian Pounds (EGP)
ℹ️ About page showcasing the brand story and mission
📞 Contact page with:
Contact form
Embedded Google Maps location
🔐 Login page user interface
📱 Mobile-friendly navigation and responsive layout
🎨 Consistent modern design across all pages
